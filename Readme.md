# Hosts

A command line program with shortcuts for managing hosts file entries.

## Usage

    Usage:
      hosts add <ip> <hostname>
      hosts remove <hostname>
      hosts list [<search string>]
      hosts edit
      hosts file

For full usage, run:

    hosts help

or

    hosts help <command name>
